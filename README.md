## Permission-Platform
### Permission-Platform简介
Permission-Platform(权限管理平台)是一个权限分配的管理平台，可接入外部应用平台，针对该应用给其平台的用户分配相关的权限，提供相应的外部接口给应用平台调用，获取该用户在其平台的权限信息。
